# Robust Chat Filter 🛡️

Это легковесный и быстрый классификатор для автоматической модерации русскоязычных чатов в реальном времени. Модель выявляет **токсичность, нецензурную лексику и спам**.

Мной был выбрал нестандартный подход: используется посимвольная сверточная нейронная сеть _(Char-level CNN)_. Это делает модель устойчивой к **намеренным опечаткам, замене букв символами/цифрами** и нестандартному интернет-сленгу, так как архитектура не привязана к фиксированному словарю токенов

## 🧠 Архитектура

- Char-CNN: блоки одномерных сверток и пулинга
- Multi-Label Classification
- Custom Loss: была реализована другая ф-ия потерь, которая динамически снижает штрафы между toxic и obscenity. Так было сделано из-за дико кривой разметки датасета

## 📂 Источники данных

Планируется обучить модель на разных неформальных датасетиках:

- OK.ru Toxic Comments
- 2ch/Pikabu Dataset
- Telegram Spam
- **DeepSeek**

Да, без шуток, я еще решил сгенерировать некоторые данные при помощи **LLM**. Получилось неплохо. Было добавлено $3000$ качественных и размеченных строк.

## 📊 Результаты

Есть высокая стабильность. Переобучения нет. Test Loss примерно 0.07, при обучении val loss и train loss дошли до похожих значений.

Основные метрики качества на отложенной тестовой выборке:

- **Spam:** F1-Score = 0.96
- **Toxic:** F1-Score = 0.89
- **Obscenity:** F1-Score = 0.74 _(тут recall = 0.86, модель старается не пропустить мат и это норм, так как датасет кривой)_
