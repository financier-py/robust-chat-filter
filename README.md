# Robust Chat Filter 🛡️

Тут будет реализован классификатор для выявления токсичности, нецензурной лексики и спама на русском языке.
Можно будет использовать как легковесную модель для автоматической модерации чатов в реальном времени.

Мной был выбран нестандартный подход: сверточная нейронная сеть будет посимвольной. 

📂 Источники данных

Планируется обучить модель на разных неформальных датасетиках:

- OK.ru Toxic Comments
- 2ch/Pikabu Dataset
- Telegram Spam
